    Support & License

    Support Forum

    The primary support channel is the GoBlocks support forum on WordPress.org. Before opening a new thread:

    1. Check the Troubleshooting guide for solutions to common issues.
    2. Search the forum for existing threads on your issue.
    3. If you cannot find an answer, open a new thread with:
      • GoBlocks version (visible in Plugins → Installed Plugins)
      • WordPress version
      • PHP version (Tools → Site Health → Info → Server)
      • List of active plugins
      • A clear description of the issue and steps to reproduce it
      • Any PHP error messages from debug.log

    Response time: We aim to respond to all forum threads within 1–2 business days.

    Bug Reports

    Found a bug? Please report it on the support forum with a clear reproduction case. For security vulnerabilities, do not post publicly — see Security → Reporting a Vulnerability.

    When reporting a bug, include:

    • The exact steps to reproduce the issue
    • What you expected to happen
    • What actually happened
    • Your environment (WordPress, PHP, browser versions)
    • Any JavaScript console errors or PHP error log entries

    Feature Requests

    Feature requests are welcome on the support forum. Tag your thread with "Feature Request" in the title. We review all requests and consider them for the roadmap, though we cannot commit to specific timelines.

    Contributing

    GoBlocks is open source and contributions are welcome. The source code is available at github.com/godevsltd/goblocks.

    Development Setup

    git clone https://github.com/godevsltd/goblocks.git
cd goblocks
composer install
npm install
npm run build

    See Developer Guide → Build from Source for full instructions.

    Contribution Guidelines

    • Open an issue before starting significant work to align on approach.
    • Code must pass PHP_CodeSniffer (WPCS) and PHPStan Level 8.
    • JavaScript/TypeScript must pass ESLint with the project configuration.
    • Include test coverage for new features.
    • All strings must use the goblocks text domain.

    Leave a Review

    If GoBlocks is helping your projects, please leave a review on WordPress.org — it helps other users discover the plugin and supports continued development.
