    Hooks & Filters

    GoBlocks provides 2 action hooks and 4 filter hooks for extending plugin behaviour without modifying plugin files.

    Action Hooks

    goblocks_block_registered

    do_action( string $block_name, array $block_args );

    Fired immediately after each GoBlocks block type is registered with register_block_type().

    ParameterTypeDescription
    $block_namestringThe registered block name, e.g. goblocks/group
    $block_argsarrayThe arguments passed to register_block_type()

    Example — log all block registrations:

    <?php
add_action( 'goblocks_block_registered', function( string $name, array $args ): void {
    error_log( 'Registered block: ' . $name );
}, 10, 2 );

    Hook priority: Fires on init at priority 10 (after WordPress's own block registration).

    goblocks_css_generated

    do_action( string $css, int $post_id, string $file_path );

    Fired after a new per-page CSS file has been written to disk.

    ParameterTypeDescription
    $cssstringThe full CSS string that was written
    $post_idintThe ID of the post the CSS was generated for
    $file_pathstringAbsolute path to the written CSS file

    Example — invalidate a CDN cache after CSS regeneration:

    <?php
add_action( 'goblocks_css_generated', function( string $css, int $post_id, string $file_path ): void {
    $url = get_permalink( $post_id );
    cloudflare_purge_url( $url );
}, 10, 3 );

    Filter Hooks

    goblocks_dynamic_tags

    apply_filters( 'goblocks_dynamic_tags', array $tags ): array;

    Allows registering custom dynamic content tags or overriding existing ones. The filter receives the full associative array of registered tags — key is the tag slug, value is a callable.

    ParameterTypeDescription
    $tagsarrayAssociative array of slug => callable. Each callable receives (array $options, WP_Post $post) and must return string.

    Example:

    <?php
add_filter( 'goblocks_dynamic_tags', function( array $tags ): array {
    $tags['my_tag'] = function( array $options, WP_Post $post ): string {
        return esc_html( get_post_meta( $post->ID, 'my_meta_key', true ) );
    };
    return $tags;
} );

    See Dynamic Content → Registering Custom Tags for a full example.

    goblocks_query_args

    apply_filters( 'goblocks_query_args', array $args, array $block_attributes ): array;

    Modifies the WP_Query arguments before the Query block executes its query.

    ParameterTypeDescription
    $argsarrayThe full WP_Query arguments array
    $block_attributesarrayThe Query block's attributes as saved in the editor

    Example — add sticky posts to every query:

    <?php
add_filter( 'goblocks_query_args', function( array $args, array $attrs ): array {
    $args['ignore_sticky_posts'] = false;
    return $args;
}, 10, 2 );

    goblocks_block_css

    apply_filters( 'goblocks_block_css', string $css, array $block, WP_Post $post ): string;

    Allows appending to or replacing the CSS generated for a specific block instance. Called once per block during the CSS generation pass.

    ParameterTypeDescription
    $cssstringThe CSS string generated by GoBlocks for this block
    $blockarrayThe parsed block array (blockName, attrs, innerBlocks)
    $postWP_PostThe post the CSS is being generated for

    Example — append a hover CSS rule to all Button blocks:

    <?php
add_filter( 'goblocks_block_css', function( string $css, array $block, WP_Post $post ): string {
    if ( $block['blockName'] !== 'goblocks/button' ) {
        return $css;
    }
    $uid = esc_attr( $block['attrs']['uid'] ?? '' );
    $css .= ".gb-block-{$uid}:hover { transform: translateY(-2px); }";
    return $css;
}, 10, 3 );

    goblocks_settings_defaults

    apply_filters( 'goblocks_settings_defaults', array $defaults ): array;

    Modifies the default settings values used when goblocks_settings is not yet set in the database (fresh install) or when a key is missing.

    ParameterTypeDescription
    $defaultsarrayThe default settings array (all 8 settings keys with their built-in defaults)

    Example — change the default container width for a theme that uses 1440px:

    <?php
add_filter( 'goblocks_settings_defaults', function( array $defaults ): array {
    $defaults['container_width'] = 1440;
    return $defaults;
} );
