    What Makes GoBlocks Different

    Design Token System
    Three-tier CSS custom property architecture. Restyle every block by overriding a handful of --gb-* tokens — no PHP or JS changes needed.
    Per-Page Static CSS
    One CSS file per page, written to disk, served with browser cache headers. Zero PHP execution for CSS on repeat page loads. Delta regeneration only rewrites changed blocks.
    Zero JS Static Blocks
    Layout, text, image, icon, button — no frontend JavaScript. Interactive blocks ship tiny vanilla IIFEs (~500 B) loaded only when the block is on the page.
    Full Site Editing
    All blocks available in the Site Editor. Dynamic content tags resolve in template context — build reusable post card templates once, use everywhere.
    Dynamic Content Tags
    22 built-in tags using {tag|option:value} syntax. Show post title, featured image, author, categories, site name — anywhere in any block.
    WCAG 2.1 AA
    ARIA patterns for every interactive block. Keyboard navigation, focus management, reduced-motion support, semantic HTML throughout.
