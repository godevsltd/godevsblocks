    Changelog

    All notable changes to GoBlocks are documented here. GoBlocks follows Semantic Versioning. Security patches are released immediately as patch versions.

    1.0.0 — 2026-07-01

    Initial Release

    The first public release of GoBlocks on WordPress.org.

    36 New Blocks

    Layout

    • Group — Flexbox and CSS Grid container with any semantic HTML tag, background, border, shadow, and link mode
    • Column — Independent column block for use inside a Group block
    • Container — Max-width centering wrapper aligned to the global container width setting

    Content

    • Text — Rich-text paragraph with dynamic content tag support
    • Heading — h1–h6 heading with gradient text support and dynamic content tags
    • Button — CTA button/anchor with icon support, hover states, and ARIA label
    • Image — Responsive image with dynamic src/alt, lazy loading, srcset/sizes
    • Icon — Inline SVG icon with 112-icon built-in library across 13 categories
    • Shape Divider — SVG decorative shapes (wave, triangle, slant, curve, zigzag) for section transitions
    • Separator — Styled horizontal rule with width, thickness, color, and style
    • Spacer — Fixed-height vertical spacer with per-breakpoint height

    Interactive

    • Tabs + Tab Panel — ARIA tablist/tab/tabpanel pattern; arrow key navigation; deep-linkable via hash
    • Accordion + Accordion Item — Native <details>/<summary>; optional FAQ Schema; no JavaScript needed
    • Modal — <dialog>-based; focus trap; Escape closes; focus returned to trigger on close
    • Flip Card — Front/back flip on hover or click; horizontal or vertical direction

    Media

    • Video — YouTube, Vimeo, and self-hosted; lazy facade poster improves LCP
    • Lottie Animation — JSON animation player with IntersectionObserver trigger
    • Slider + Slide — Touch-enabled carousel; autoplay; responsive slides-per-view

    Query & Archive

    • Query — Visual post query builder: post type, taxonomy, author, date range, order, exclusions
    • Query Loop — Loop template container; all dynamic content tags supported
    • No Results — Fallback content for empty query results
    • Pagination — Standard, numbered, Load More (Ajax), and Infinite Scroll modes
    • Navigation — Full site nav from a WordPress menu; hamburger mobile toggle; submenus with aria-expanded

    Showcase

    • Counter — Animated counting number with IntersectionObserver trigger and aria-live
    • Progress Bar — Animated width fill with role="progressbar" ARIA
    • Alert — Info / Success / Warning / Error variants with semantic ARIA roles
    • Star Rating — 0–5 star display with optional Schema.org AggregateRating output
    • Countdown — Live countdown timer; expired action: hide, message, or redirect
    • Pricing Card — Plan name, price, billing period, feature list with CTA button and badge
    • Social Share — Share links for Twitter/X, Facebook, LinkedIn, Pinterest, WhatsApp, Telegram, Email, Copy Link
    • Table of Contents — Auto-generated page TOC from headings; sticky mode; smooth scroll
    • Timeline + Timeline Item — Vertical timeline with alternating or single-column layout

    41 Built-in Patterns

    Hero (3), Features / How-It-Works (2), Cards (2), Pricing (1), FAQ (1), Stats (1), CTA (1), Blog (2), Portfolio (1), Team (1), Testimonials (2), Newsletter (1), Contact (1), Logo Cloud (1).

    Core Systems

    • Per-page CSS generation — Static CSS file per post; delta regeneration via block fingerprinting; cache-busting version query string
    • CSS Design Token System — Three-tier --gb-* CSS custom property architecture: primitive / semantic / component
    • Dynamic Content Tag System — 22 tags across Post Data, Author, Featured Image, Taxonomy, Site, and User categories; custom tag registration via goblocks_dynamic_tags filter
    • Six-Breakpoint Responsive Controls — xs/sm/md/lg/xl/2xl; mobile-first; sync inheritance mode
    • Global Styles — Color palette and typography presets registered as CSS custom properties and WordPress editor presets
    • REST API — 14 endpoints under /wp-json/goblocks/v1/ for settings, styles, CSS cache, patterns, and blocks
    • WP-CLI commands — Cache clear, CSS regeneration, settings export/import
    • WCAG 2.1 AA — Full ARIA implementation on all interactive blocks; keyboard navigation; focus management; prefers-reduced-motion support
    • Full Site Editing — All blocks and patterns available in the Site Editor; theme.json integration; template-level CSS generation
    • Multisite compatible — Independent CSS cache and settings per sub-site
    • WordPress Plugin Check (PCP) compliant — Passes all automated WP.org plugin review checks
