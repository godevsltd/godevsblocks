    Blocks Reference

    GoBlocks registers 36 blocks under the goblocks namespace. All blocks are available in the GoBlocks category in the block inserter and in the Site Editor.

    Layout Blocks

    Group goblocks/group

    The primary layout container. Wraps child blocks in a <div>, <section>, <article>, <aside>, <header>, <footer>, or <main> element. Supports both Flexbox and CSS Grid layout modes.

    • Configurable semantic HTML tag
    • Flex: direction, wrap, alignment, justify-content, gap
    • Grid: columns, rows, auto-flow, gap
    • Background: solid color, gradient, image with overlay
    • Border, border-radius, box shadow, full responsive control on all spacing
    • Link mode: wraps the entire block in an <a> tag
    • Animation: fade-in, slide-up, etc. via data attribute

    Column goblocks/column

    A single column for use inside a Group block in flex or grid mode. Controls its own width, padding, and alignment independently of sibling columns.

    • Column span in grid context
    • Flex grow / shrink / basis
    • Independent background, border, padding

    Container goblocks/container

    A max-width centering wrapper. Renders a <div> with max-width set to the Container Width from Settings (default: 1200px) and margin: 0 auto. Use inside a full-width Group to center page content.

    • Respects the global container width setting
    • Can be overridden per-block for narrow or wide containers
    • Optional padding override

    Content Blocks

    Text goblocks/text

    A rich-text paragraph block with full design token integration, dynamic content support, and typography tokens from the Global Styles panel. Supports dynamic content tags in content.

    Heading goblocks/heading

    Semantic heading block supporting h1 through h6. All standard Typography controls plus gradient text support and dynamic content tags.

    • Heading level selector (h1–h6)
    • Gradient text via background-clip: text
    • Dynamic content in the heading text
    • Per-breakpoint font-size overrides

    Button goblocks/button

    A CTA button or anchor link. Renders as <a> (for links) or <button> (for non-navigational actions). Full styling control with hover state support via CSS custom properties.

    • Href, target, rel attributes
    • Primary / secondary / outline style variants
    • Icon support (left or right of label)
    • ARIA label for accessibility

    Image goblocks/image

    A responsive image block with dynamic content support for src and alt, making it possible to show the post's featured image inside a Query Loop.

    • Dynamic src from {featured_image} tag
    • Dynamic alt text
    • Object fit / object position, responsive width and aspect ratio
    • Outputs loading="lazy", srcset, and sizes automatically

    Icon goblocks/icon

    An inline SVG icon block with a built-in visual picker of 112 icons across 13 categories: UI, Arrows, Communication, People, Files, Media, Commerce, Location, Sharing, Social, Time, Misc.

    • Stroke / fill / both color controls
    • Configurable size (responsive)
    • Optional link wrapping

    Shape Divider goblocks/shape

    Decorative SVG shapes for section transitions. Positioned at the top or bottom edge of a container.

    • Multiple shape presets: wave, triangle, slant, curve, zigzag
    • Flip horizontally / vertically
    • Responsive height

    Separator goblocks/separator

    A styled horizontal rule (<hr>) with more configuration than WordPress's core Separator: width, thickness, color, and style (solid, dashed, dotted, double).

    Spacer goblocks/spacer

    A fixed-height vertical spacer with per-breakpoint height, so it can collapse on mobile or expand on desktop. Min-height / max-height constraints supported.

    Interactive Blocks

    Tabs goblocks/tabs

    An ARIA-compliant tabbed interface following the ARIA Tabs Pattern. Arrow keys move between tabs, Home/End jump to first/last, Enter/Space activates.

    • Any number of tabs
    • Orientation: horizontal or vertical
    • Alignment: left, center, right
    • Custom active tab color tokens
    • Deep-linkable via URL hash

    Generates role="tablist", role="tab", role="tabpanel", aria-selected, aria-controls, aria-labelledby automatically.

    Tab Panel goblocks/tab-panel

    A single panel within a Tabs block. Contains any child blocks. Insert Tab Panel blocks directly inside the Tabs block via the block inserter.

    Accordion goblocks/accordion

    A disclosure widget built on native <details>/<summary> HTML elements — no JavaScript required for the expand/collapse behavior.

    • Native <details>/<summary> — works without JS
    • Optional FAQ schema markup (schema.org/FAQPage)
    • Animation on open/close via CSS transition
    • Allow one open at a time (exclusive) mode

    SEO: Enable FAQ Schema in the block Inspector to output structured data that Google can show as rich results in search.

    Accordion Item goblocks/accordion-item

    A single question/answer pair within an Accordion. The summary (question) is rendered as <summary>, the content (answer) can contain any child blocks.

    Modal goblocks/modal

    A dialog overlay triggered by a button click. Renders content inside a <dialog> element with proper focus management.

    • Trigger button fully customizable
    • Backdrop click closes modal
    • Keyboard: Escape closes, Tab is trapped inside while open
    • Accessible: role="dialog", aria-modal="true", aria-labelledby

    Flip Card goblocks/flip-card

    A card with a front face and a back face that flips on hover or click.

    • Flip direction: horizontal or vertical
    • Trigger: hover or click
    • Independent front/back backgrounds and content
    • ARIA toggle for keyboard users

    Media Blocks

    Video goblocks/video

    A video embed block supporting YouTube, Vimeo, and self-hosted MP4/WebM. Includes a lazy-load facade (poster image) that defers the video iframe until user interaction, improving Core Web Vitals.

    • YouTube and Vimeo embed support (auto-extracts ID from URL)
    • Self-hosted video via <video> element
    • Lazy facade: shows poster image, loads iframe on click
    • Aspect ratio control, autoplay, loop, mute (self-hosted)

    Lottie Animation goblocks/lottie

    Renders a Lottie JSON animation file hosted on your server or via URL.

    • URL or media library JSON source
    • Autoplay, loop, speed controls
    • Play on viewport entry (IntersectionObserver)
    • Responsive width

    Slider goblocks/slider

    A touch-enabled slide carousel.

    • Autoplay with configurable interval
    • Navigation arrows and dot indicators (optional)
    • Responsive slides-per-view
    • Loop mode, keyboard navigation between slides

    Slide goblocks/slide

    A single slide within a Slider block. Contains any child blocks. Insert Slide blocks directly inside the Slider block.

    Query & Archive Blocks

    Query goblocks/query

    A visual post query builder replacing WordPress's core Query Loop block with a richer filtering interface. Configure the query entirely from the block Inspector — no code required.

    Filter options: Post type, Taxonomy / term, Author, Date range, Post status, Posts per page, Orderby and order, Exclude current post.

    Fully customizable via the goblocks_query_args filter. See Hooks & Filters.

    Query Loop goblocks/query-loop

    The loop template rendered inside the Query block. Contains any child blocks — typically Image, Heading, Text, and Button blocks using dynamic content tags.

    {post_title}            — displays the title of each post
{post_date|format:M j, Y} — displays the date formatted
{post_excerpt}          — displays the excerpt
{featured_image|size:medium} — displays the featured image

    No Results goblocks/query-no-results

    Fallback content shown when the Query block returns zero posts. Contains any child blocks. Typical use: a friendly "No posts found" message with a search form.

    Pagination goblocks/pagination

    Pagination for the Query block with four modes:

    ModeDescription
    StandardPrevious / Next page links
    NumberedPage number links (1, 2, 3 …)
    Load MoreAjax button that appends posts without a full reload
    Infinite ScrollPosts load automatically as the user scrolls

    Navigation goblocks/navigation

    A full site navigation menu block with mobile hamburger toggle, mega menu support, and submenu chevron indicators.

    • Renders from a WordPress nav menu
    • Responsive: hamburger toggle on mobile
    • Submenu support with aria-expanded toggle
    • Sticky / fixed positioning option
    • Logo + menu + CTA button layout

    Showcase Blocks

    Counter goblocks/counter

    An animated number counter that counts up when it enters the viewport via IntersectionObserver.

    • Start and end values, duration, prefix and suffix (e.g., $ / + / %)
    • Easing: linear or ease-out
    • aria-live="polite" announces final value to screen readers

    Progress Bar goblocks/progress-bar

    An animated horizontal progress bar that animates on viewport entry.

    • Configurable percentage value, label (inside or above), color (solid or gradient)
    • ARIA: role="progressbar", aria-valuenow, aria-valuemax

    Alert goblocks/alert

    A contextual notification banner with four variants: Info, Success, Warning, Error. Each maps to semantic color tokens and an appropriate ARIA role (role="alert" for errors).

    • Icon (optional, auto-matched to variant)
    • Dismissible (close button) mode
    • Bordered or filled style

    Star Rating goblocks/star-rating

    A visual star rating display (read-only). Rating value from 0.0 to 5.0 with half-star support. Optional Schema.org AggregateRating structured data output.

    Countdown goblocks/countdown

    A live countdown timer to a target date and time.

    • Target date/time picker with timezone support
    • Display: days, hours, minutes, seconds (any combination)
    • Expired action: hide block, show message, or redirect URL
    • Labels for each unit (localizable)

    Pricing Card goblocks/pricing

    A structured pricing tier card with plan name, price, billing period, feature list with checkmarks, highlighted / featured tier style, built-in CTA button, and "Most popular" badge.

    Social Share goblocks/social-share

    Generates social sharing links for the current page. Supported platforms: Twitter/X, Facebook, LinkedIn, Pinterest, WhatsApp, Telegram, Email, Copy Link.

    • Icon-only, icon + label, or label-only style
    • Circle or rounded-square icon shape
    • URL is dynamically resolved from the current post permalink

    Table of Contents goblocks/table-of-contents

    Auto-generates a linked table of contents from the headings on the current page.

    • Heading level range (e.g., h2–h4 only)
    • Smooth scroll to anchor, sticky position mode
    • Collapsible on mobile, numbered or bulleted list style

    Timeline goblocks/timeline

    A vertical timeline layout with alternating left/right or single-column layout. Connector line styling: color, style, width. Contains Timeline Item child blocks.

    Timeline Item goblocks/timeline-item

    A single event in a Timeline block. Features: Date / label, Icon or marker dot, Content (any child blocks), Animation on viewport entry.
